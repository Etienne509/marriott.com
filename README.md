#  Projet Web : Marriott Port-au-Prince
##  Présentation
Projet réalisé dans le cadre de mes études en Génie Informatique. C'est un site vitrine premium et responsive pour le Marriott Port-au-Prince, rendu dynamique grâce à JavaScript.
##  Structure du Site
 * **Accueil (index.html) :** Vitrine avec message dynamique, formulaire de réservation et statistiques.
 * **À Propos (about.html) :** Histoire, valeurs et ancrage local de l'hôtel.
 * **Services (services.html) :** Présentation des prestations (piscine, sécurité, Wi-Fi).
 * **Projets (projects.html) :** Focus sur l'infrastructure moderne à Turgeau.
 * **Contact (contact.html) :** Formulaire pour envoyer une demande sécurisée.
##  Fonctionnement & Interaction (JavaScript)
 * **Visuel :** Écran de chargement professionnel et bouton de thème (Sombre / Clair) qui adapte les couleurs du texte automatiquement.
 * **Réservation Intelligente :** Calcul de prix en temps réel selon la suite choisie et mise à jour immédiate d'un tableau de bord de statistiques.
 * **Sauvegarde :** Stockage local (localStorage) pour garder les réservations affichées même après actualisation.
 * **Formulaire de Contact :** Interception par JavaScript pour une validation fluide avec affichage de notifications modernes (Toasts).
